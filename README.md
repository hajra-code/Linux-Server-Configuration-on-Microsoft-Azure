# Linux Server Configuration on Microsoft Azure

## Project Overview

This project demonstrates the deployment and configuration of multiple network services on an Ubuntu Virtual Machine hosted on Microsoft Azure.

The objective of this project was to gain practical experience in Linux system administration, cloud computing, and network service configuration.


## Services Configured

- FTP Server (VSFTPD)
- Apache Web Server
- DNS Server (BIND9)
- DHCP Server (ISC DHCP)
- Azure Network Security Group (NSG)
- SSH Remote Access


## Technologies Used

- Ubuntu Linux
- Microsoft Azure
- VSFTPD
- Apache2
- BIND9
- ISC DHCP Server
- FileZilla
- SSH


## Project Workflow

1. Created an Ubuntu Virtual Machine on Microsoft Azure.
2. Connected to the server using SSH.
3. Installed and configured VSFTPD for secure file transfer.
4. Created FTP users and configured permissions.
5. Configured Azure firewall rules for FTP access.
6. Installed and configured Apache Web Server.
7. Created and hosted a custom HTML webpage.
8. Installed and configured the BIND9 DNS Server.
9. Created a custom DNS zone and verified name resolution.
10. Installed and configured the ISC DHCP Server.
11. Tested all services successfully.
